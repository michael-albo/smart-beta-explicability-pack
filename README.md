# smart-beta-explicability-pack
Smart-Beta &amp; ETF “Explainability Pack”
Les modèles d’attribution traditionnels fournissent une moyenne historique des sensibilités (β), mais pas les règles dynamiques qui expliquent quand et pourquoi ces expositions produisent (ou détruisent) de l’alpha.
Nous allons construire un “Smart-Beta Explainability Pack” qui répond aux questions en décomposant la performance selon trois axes :
* Attribution par facteur : d’où vient le rendement ?
* Alpha hygiene : ce qui reste après avoir expliqué par les facteurs
* Stabilité des expositions : dans quelles conditions ces relations tiennent ?
